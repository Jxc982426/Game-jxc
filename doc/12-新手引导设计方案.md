# 12-新手引导设计方案

## 📋 文档概述

本文档描述了游戏的新手引导系统，包括教程流程、交互设计、难度曲线和激励机制。帮助新玩家快速上手并理解游戏核心玩法。

**创建日期：** 2026-04-02  
**适用版本：** v1.0  
**目标用户：** 首次接触 ARPG/挂机游戏的玩家

---

## 一、引导设计理念

### 1.1 核心原则

```
✅ 渐进式教学：一次只教一个概念
✅ 实践为主：边做边学，而非纯文本说明
✅ 即时反馈：玩家操作后立即给出反馈
✅ 可跳过：老玩家可以跳过引导
✅ 可回顾：随时查看已学内容
```

### 1.2 引导阶段划分

| 阶段 | 名称 | 时长 | 目标 |
|------|------|------|------|
| Phase 1 | 初入世界 | 3 分钟 | 基础移动和攻击 |
| Phase 2 | 战斗入门 | 5 分钟 | 技能使用和战斗循环 |
| Phase 3 | 收获成长 | 5 分钟 | 拾取装备和属性提升 |
| Phase 4 | 挂机解放 | 3 分钟 | 开启自动挂机 |
| Phase 5 | 自由探索 | - | 引导结束，自由活动 |

**总时长：** 约 15-20 分钟

## 二、详细引导流程

### 2.1 Phase 1 - 初入世界（3 分钟）

#### 场景设置
```
地点：新兵训练营（简化版城镇）
怪物：训练假人（不会反击）
限制：无法离开训练营区域
```

#### 步骤分解

**Step 1.1: 欢迎界面（15 秒）**
```
显示内容:
- 半透明遮罩背景
- 中央文本："欢迎来到暗黑大陆！"
- 副标题："让我们从基础开始"
- 按钮："开始训练" [高亮闪烁]

UI 效果:
- 淡入动画 (0.5s)
- 文本逐字显示
- 按钮脉冲效果
```

**Step 1.2: 移动教学（45 秒）**
```
触发条件：点击"开始训练"后

教学内容:
1. 高亮显示 WASD 键位提示
2. 箭头指向目标点（绿色圆圈标记）
3. 文本提示："使用 WASD 或方向键移动"

玩家任务：移动到目标点

完成条件：玩家到达目标点

成功反馈:
- ✅ 音效：清脆的"叮"声
- 视觉：目标点爆发出绿色粒子
- 文本："很好！继续前进"
```

**Step 1.3: 攻击教学（60 秒）**
```
触发条件：玩家到达第一个目标点

教学内容:
1. 出现训练假人（HP: 100, 不攻击）
2. 高亮鼠标左键 / J 键
3. 文本提示："点击鼠标左键攻击敌人"

玩家任务：攻击假人直到死亡

完成条件：假人 HP 归零

特殊设计:
- 第一次攻击必定暴击（教学爽快感）
- 弹出大号伤害数字："150 暴击！"
- 假人缓慢倒下（慢动作 0.5 秒）
```

**Step 1.4: 连续击杀练习（90 秒）**
```
触发条件：击败第一个假人

教学内容:
1. 同时出现 3 个假人
2. 文本提示："击败所有训练假人！"
3. 显示击杀计数：0/3

玩家任务：击败 3 个假人

进度反馈:
- 每击败一个：✅ +1 计数
- 全部击败：🎉 "太棒了！"
```

```

- 技能重新可用
- 文本："法力已恢复！"
```

**Step 2.4: 连招练习（120 秒）**
```

- 击败 5 个活木桩

提示系统:
- 如果只用普攻："试试使用重击！"
```

## 三、引导系统设计

### 3.1 提示系统架构

```gdscript
# TutorialManager.gd
extends Node

signal tutorial_started(tutorial_id: String)
signal tutorial_completed(tutorial_id: String)

var current_tutorial: String = ""
var completed_tutorials: Array[String] = []
var tutorial_states: Dictionary = {}

# 提示类型
enum HintType {
    TOOLTIP,      # 悬停提示
    HIGHLIGHT,    # 高亮显示
    ARROW,        # 箭头指引
    DIALOG,       # 对话框
    DEMO          # 演示动画
}

# 创建提示
func show_hint(hint_type: HintType, target: Node, text: String):
    match hint_type:
        HintType.TOOLTIP:
            create_tooltip(target, text)
        HintType.HIGHLIGHT:
            create_highlight(target, text)
        HintType.ARROW:
            create_arrow(target, text)
        HintType.DIALOG:
            create_dialog(text)
        HintType.DEMO:
            play_demo(target)

# 检查完成条件
func check_completion(condition: Dictionary) -> bool:
    match condition.type:
        "reach_position":
            return player.position.distance_to(condition.target) < 5.0
        "kill_enemy":
            return kill_count >= condition.count
        "use_skill":
            return skill_use_count[condition.skill_id] >= condition.count
        "equip_item":
            return player.equipment.has(condition.item_type)
    return false
```

### 3.2 交互式提示组件

```gdscript
# InteractiveHint.gd
class_name InteractiveHint
extends Control

@export var target_node: NodePath
@export var hint_text: String
@export var arrow_offset: float = 50.0

var arrow_position: Vector2 = Vector2.ZERO
var pulse_timer: float = 0.0

func _draw():
    # 绘制箭头指向目标
    if target_node:
        var target_pos = get_global_mouse_position()
        draw_arrow(arrow_position, target_pos, Color.YELLOW)
    
    # 绘制文本框
    draw_hint_box(hint_text)

func draw_arrow(from: Vector2, to: Vector2, color: Color):
    var direction = (to - from).normalized()
    var arrow_length = 20.0
    
    # 箭杆
    draw_line(from, to - direction * arrow_length, color, 3.0)
    
    # 箭头
    var left = to - direction * arrow_length + Vector2(-direction.y, direction.x) * 10.0
    var right = to - direction * arrow_length + Vector2(direction.y, -direction.x) * 10.0
    draw_triangle(to, left, right, color)
```

```gdscript
# 新手引导进度
var tutorial_progress = {

            {"id": "combo_practice", "completed": false}
        ],
        "reward": {"exp": 500, "gold": 100}
    }
    # ... 更多阶段
}

func get_completion_percentage() -> float:
    var total_steps = 0
    var completed_steps = 0
    
    for phase in tutorial_progress.values():
        for step in phase.steps:
            total_steps += 1
            if step.completed:
                completed_steps += 1
    
    return (completed_steps / total_steps) * 100.0
```

## 五、跳过与回顾机制

### 5.1 跳过条件

```gdscript
# 允许跳过的条件
func can_skip_tutorial() -> bool:
    # 账号等级 > 10
    if account_level > 10:
        return true
    
    # 已有角色达到 20 级
    if has_character_above_level(20):
        return true
    
    # 累计游戏时间 > 5 小时
    if total_playtime > 5 * 3600:
        return true
    
    # 手动输入跳过指令（隐藏功能）
    if Input.is_action_just_pressed("skip_tutorial"):
        return true
    
    return false
```
