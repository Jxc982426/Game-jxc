# 10-美术资源技术规范

## 📋 文档概述

本文档定义了游戏开发中所有美术资源的技术规格，包括精灵尺寸、帧率、压缩格式、纹理图集等标准。确保美术资源在 Godot 引擎中的高效使用。

## 一、2D 精灵规格

### 1.1 角色精灵

#### 玩家角色

| 状态 | 尺寸（像素） | 帧数 | 帧率 | 总时长 | 文件格式 |
|------|-------------|------|------|--------|----------|
| 待机 | 64×64 | 4 | 8 FPS | 0.5s | PNG |
| 移动 | 64×64 | 8 | 12 FPS | 0.67s | PNG |
| 攻击 - 剑 | 96×64 | 6 | 15 FPS | 0.4s | PNG |

## 三、纹理压缩规范

### 3.1 压缩格式选择

| 平台 | BGR 压缩 | Alpha 压缩 | 法线贴图 |
|------|---------|-----------|---------|
| Windows | DXT5 | DXT5 | BC5 |
| macOS | PVRTC | PVRTC | PVRTC |
| Linux | ETC2 | ETC2 | EAC |
| Web | BasisU | BasisU | BasisU |

### 3.2 Godot 导入设置

```gdscript
# .import 文件配置示例（带 Alpha 的纹理）
[remap]
Importer = "texture"
Type = "StreamTexture"
Path = "res://.import/texture.png.hash"

[deps]
Source = "res://textures/texture.png"

[params]
compress/mode = 2  # VRAM 压缩
compress/high_quality = true
compress/lossy_quality = 0.7
compress/hdr_compression = 1
compress/normal_map = false
compress/channel_pack = 0
flags/repeat = 0
flags/filter = true
flags/mipmaps = false
```

### 3.3 纹理图集（Texture Atlas）

**合并规则：**
- 相同材质的精灵优先合并
- 动画帧打包到同一图集
- 图集尺寸：2 的幂次方（512, 1024, 2048）
- 间距：至少 2 像素

**示例布局：**
```
Atlas: player_sprites_atlas.png (1024×1024)
├── idle 动画：4 帧 (64×64 each)
├── run 动画：8 帧 (64×64 each)
├── attack 动画：6 帧 (96×64 each)
└── ...其他
```

### 5.2 Shader 特效

#### 溶解效果

```glsl
// dissolve.shader
shader_type canvas_item;

uniform sampler2D noise_texture;
uniform float dissolve_amount : hint_range(0, 1) = 0.0;
uniform vec4 edge_color : hint_color = vec4(1.0, 0.5, 0.0, 1.0);

void fragment() {
    vec4 tex_color = texture(TEXTURE, UV);
    float noise = texture(noise_texture, UV).r;
    
    if (noise < dissolve_amount) {
        discard;
    } else if (noise < dissolve_amount + 0.1) {
        COLOR = edge_color;
    } else {
        COLOR = tex_color;
    }
}
```

#### 发光效果

```glsl
// glow.shader
shader_type canvas_item;

uniform vec4 glow_color : hint_color = vec4(0.0, 1.0, 1.0, 1.0);
uniform float glow_intensity = 1.5;
uniform float glow_radius = 0.2;

void fragment() {
    vec4 tex_color = texture(TEXTURE, UV);
    float alpha = tex_color.a;
    
    // 简单的发光效果
    vec3 final_color = tex_color.rgb + (glow_color.rgb * glow_intensity * alpha);
    
    COLOR = vec4(final_color, alpha);
}
```

## 六、性能优化

### 6.1 纹理内存管理

```gdscript
# 纹理内存预算
var texture_memory_budget = {
    "character_sprites": "50 MB",
    "enemy_sprites": "30 MB",
    "item_icons": "20 MB",
    "environment": "80 MB",
    "ui_elements": "20 MB",
    "effects": "20 MB",
    "total": "220 MB"
}

# 动态加载/卸载
var loaded_textures: Dictionary = {}

func load_texture_on_demand(path: String):
    if not loaded_textures.has(path):
        var texture = ResourceLoader.load(path)
        loaded_textures[path] = texture
    
    return loaded_textures[path]

func unload_unused_textures():
    var used_paths = get_currently_used_paths()
    for path in loaded_textures.keys():
        if not used_paths.has(path):
            loaded_textures[path].take_over_path()
            loaded_textures.erase(path)
```

### 6.2 绘制调用优化

**合批原则：**
1. 使用相同材质的精灵自动合批
2. 避免频繁切换材质
3. 使用 Texture Atlas 减少材质数量
4. Z-index 连续的精灵更容易合批

**Godot 设置：**
```gdscript
# 项目设置 → Rendering → Batching
use_batching = true
batching_options = {
    "max_vertices": 32767,
    "max_indices": 65535,
    "use_gpu_batching": true
}
```
